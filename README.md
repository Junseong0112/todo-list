## 기능

1. OAuth를 이용하여 깃허브, 구글 로그인으로 가능
2. 이메일로 가입 기능 구현
3. Supabase를 이용하여 데이터베이스 연결
4. 할일 생성, 삭제, 체크 까지 구현
